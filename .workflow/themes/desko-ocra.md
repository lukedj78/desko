# Desko Ocra

Il tema originale Desko. Palette ispirata al sole milanese, ocra caldo (#E8B931) come primary, ambient grigio caldo per backgrounds, font Inter ovunque + JetBrains Mono per code/metadata.

## Quando usarlo

- Default per nuove installazioni Desko
- Brand "warm tech" — software interno che non sembra né enterprise sterile né startup chiassoso

## Identità

- Ocra = energia, calma, espressione (Milano, sole)
- Geometrie morbide (radius 0.5rem)
- Tipografia letta, no decorativo
