# Nordic Minimal

Variante minimalista — palette monocromatica calda (grigi neutri), primary slate scuro, geometrie morbide (radius 12px), font Geist ovunque per look pulito tipo Vercel / Linear.

## Quando usarlo

- Aziende design-conscious che premiano la sobrietà
- Settori che lavorano molto con dati / report (no distrazioni cromatiche)
- Brand minimalista, "less is more"

## Identità

- Monocromia = focus sul contenuto
- Geist = font moderno, ottimizzato per UI
- Radius generoso = comfort visivo, no aggressività
